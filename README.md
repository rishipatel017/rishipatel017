### 🔐 StegSafe ShadowEncode — Steganography & Encryption Tool
> **React · TypeScript · Flask · Python · AES-256-GCM · Docker**

A sophisticated web-based security tool combining AES-256 encryption (PBKDF2 key derivation) with LSB steganography to hide encrypted data inside digital images. Offers 1-bit, 2-bit, and 4-bit encoding modes, PSNR visual analysis, histogram comparison, and batch processing — all stateless with no server-side data storage.
